Search Thru!

A local search engine router

Use DuckDuckGo/Kagi !bangs to search directly on any of 10 000+ supported websites. For example, caesar !w opens Julius Caesar's Wiki page, !pin bauhaus leads to 1920’s Pinterest, and cute cats !yt should speak for itself. Bangs also work as bookmarks for homepages: use !r, !a, !gh, !x to open Reddit, Amazon, Github, and X.

Fast.

Thru! redirects your query in ~0.1 second: faster than DDG, Kagi or other local routers.

Private.

Thru! works locally and have no way to store your search history — check out the source code to confirm.

Customizable.

You can change the search engine from DuckDuckGo to Google or anything else. More features are coming.

Install for Chrome, Brave, Opera...

  1. Go to Settings › Search engine › Manage search engines and site searches
  2. Scroll to Site search and click Add button
  3. Set Name and Shortcut fields to anything you want
  4. Set URL... to https://searchth.ru/?q=%s
  5. Add the search engine, click on three dots ⋮ and select Make default

Install for Firefox, Zen...

  1. Right click on URL in address bar of this page
  2. Click Add "Thru!"
  3. Go to Settings › Search and choose Thru! as a default search engine

Install for Safari

Safari does not support custom search engines, stay tuned for a browser extension.

Settings

Use !g for Google, !ya for Yandex and !b for Bing.
