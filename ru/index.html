Search Thru!

Локальный маршрутизатор поисковых запросов

Используйте !bangs от DuckDuckGo/Kagi, чтобы искать напрямую на любом из 10 000+ поддерживаемых сайтов. Например, caesar !w открывает страницу Юлия Цезаря в Википедии, !pin bauhaus ведёт к Pinterest 1920‑х, а cute cats !yt говорит само за себя. Банги также работают как закладки для главных страниц: используйте !r, !a, !gh, !x чтобы открыть Reddit, Amazon, Github и X.

Быстро.

Thru! перенаправляет запрос за ~0,1 секунды: быстрее, чем DDG, Kagi или другие локальные роутеры.

Приватно.

Thru! работает локально и не может хранить историю поиска — загляните в исходный код чтобы убедиться.

Настраиваемо.

Вы можете сменить поисковик с DuckDuckGo на Google или любой другой. Больше функций скоро.

Установка

Chrome Firefox Safari

Chrome, Яндекс.Браузер, Opera и т. д.

  1. Откройте Настройки › Поисковая система › Управление поисковыми системами и поиском по сайтам
  2. Прокрутите до Поиск по сайту и нажмите Добавить
  3. В полях Название и Ярлык укажите любые значения
  4. В поле URL… вставьте https://searchth.ru/?q=%s и нажмите Добавить
  5. Нажмите на три точки ⋮ и выберите Сделать по умолчанию

Firefox, Zen и т. д.

  1. Щёлкните правой кнопкой по адресной этой страницы
  2. Нажмите Добавить “Thru!”
  3. Перейдите в Настройки › Поиск и выберите Thru! как поисковик по умолчанию

Safari

Safari не поддерживает пользовательские поисковые системы, ждите расширение.

Настройки

Используйте !g для Google, !ya для Яндекса и !b для Bing.
